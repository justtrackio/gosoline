There are different ways to load configurations into your application:

<details>
<summary>From a file</summary>

First, configure your project in a yaml file:

<CodeBlock showLineNumbers language="yaml" title="config.dist.yml">{ConfigDist}</CodeBlock>

Then, load your configurations into your application:

<CodeBlock showLineNumbers language="go" title="main.go">{FromFile}</CodeBlock>

Here, you:

1. Import the `cfg` package.
2. Create a new configuration object.
3. Load configurations from the file.
4. Apply the options.

</details>

<details>
<summary>From a map</summary>

<CodeBlock showLineNumbers language="go" title="main.go">{FromMap}</CodeBlock>

Here, you:

1. Import the `cfg` package.
2. Create a new configuration object.
3. Load configurations from the map.
4. Apply the options.

</details>

<details>
<summary>From individual settings</summary>

<CodeBlock showLineNumbers language="go" title="main.go">{FromSetting}</CodeBlock>

Here, you've:

1. Import the `cfg` package.
2. Create a new configuration object.
3. Load configurations from settings.
4. Apply the options.

</details>

<details>
<summary>From multiple sources</summary>

<CodeBlock showLineNumbers language="go" title="main.go">{Main}</CodeBlock>

Here, you've:

1. Import the `cfg` package.
2. Create a new configuration object.
3. Load configurations from multiple sources.
4. Apply the options.

</details>

Once you've loaded your configurations, you can access them with type-specific getters. For example:

<CodeBlock showLineNumbers language="go" title="main.go" lineHighlight="22-23">{UseConfigs}</CodeBlock>
