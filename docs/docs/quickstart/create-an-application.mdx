# Create an application

For running a simple gosoline application we use the following 3 files:

- `module.go` → kernel module with business logic
- `config.dist.yml` → base configuration file
- `main.go` → setup and run the application

## Create the kernel module
The kernel module contains the actual application logic. The `HelloWorldModule` struct has to implement the
`kernel.Module` interface. The func `NewHelloWorldModule` is of the type `kernel.ModuleFactory` and is used to add
the module to the application.
